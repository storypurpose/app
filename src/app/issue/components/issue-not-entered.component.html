StoryPurpose for Jira

StoryPurpose is a set of visualization tools for agile teams that help ensure every story's purpose aligns with that of the organization
Learn more...
  • Epic/Story Proceed by entering Jira issue key
    • EpicPurpose: Useful for visualizing all EPIC issues and how they span across various components/modules within the application.
    • StoryPurpose: Useful for visualizing a selected story in the context of other linked issues, highlighting dependencies and overall associated feature delivery pipeline.
  • Jira query Proceed by entering Jira query (JQL)
    • Storyboard and timeline for related issues: This provides flexibility to choose multiple relevant issues and view on a story map.
